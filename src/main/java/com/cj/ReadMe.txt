mapConfig.txt:概率相关配置文件
account.txt:账号密码相关配置文件
itemList.txt:道具/图片相关配置文件
XXXRecord.txt：抽奖记录
userItem.txt：拥有的物品 奖品/抽奖券
costItem.txt:显示消耗奖券配置文件
exchangeList.txt：兑换配置文件
exchangeRecord.txt：兑换记录

图片配置在jar包，路径\BOOT-INF\classes\static\images

运行方式:
打开当前目录 ，CMD运行
java -Dfile.encoding=utf-8 -jar cj.jar
网页打开http://localhost:8082/cj/card/CardPage
抽奖界面的ui图标均可点击
